# 📰 Automated News Classification System

A Machine Learning-powered Streamlit web application that classifies live news headlines into **World**, **Sports**, **Business**, and **Sci/Tech** categories using **NLP**, **Scikit-learn**, and **NewsAPI**.

## 🚀 Live Demo

**🔗 Try the application here:**

https://automated-news-classification-system-hmmqpcqfkgvpzgxrs6yxpx.streamlit.app/

---

## ✨ Features

- 📡 Fetches live news using NewsAPI
- 🤖 Machine Learning-based news classification
- 🌍 Classifies news into World, Sports, Business, and Sci/Tech
- 📊 Displays prediction confidence
- 🔎 Category-based filtering
- 🎨 Interactive Streamlit user interface

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Requests
- NewsAPI
- HTML & CSS
